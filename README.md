# Problema

Os carregadores da linha GoodWe HCA G2 não possuem um sistema nativo de faturamento e cobrança e a ausência do protocolo OCPP impede que esses aparelhos sejam integrados diretamente aos aplicativos de recarga pública já existentes no mercado. Além disso, a instalação de múltiplos carregadores de alta potência traz o risco iminente de sobrecarregar a rede elétrica local e derrubar o disjuntor principal do estabelecimento nos horários de pico de consumo das lojas. Portanto, há uma necessidade clara de uma solução que viabilize o modelo de negócios desses comércios sem colocar em risco a infraestrutura elétrica local.
